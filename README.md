# Customer Churn EDA Analysis

## Overview
This project performs Exploratory Data Analysis (EDA) on customer churn data to identify trends, patterns, and factors that contribute to customer attrition. The analysis helps businesses understand customer behavior and make data-driven decisions to improve customer retention.

---

## Objectives
- Analyze customer demographics and service usage
- Identify factors affecting customer churn
- Visualize important trends and relationships
- Generate business insights from the dataset
- Prepare the data for future predictive modeling

---

## Dataset
The dataset contains customer information such as:
- Gender
- Senior citizen status
- Contract type
- Internet services
- Monthly charges
- Tenure
- Payment method
- Churn status

---

## Key Analysis Performed
- Churn distribution analysis
- Contract type vs churn
- Monthly charges analysis
- Tenure analysis
- Payment method comparison
- Correlation analysis
- Customer segmentation visualizations

---

## Visualizations
The project includes:
- Count plots
- Histograms
- Heatmaps
- Boxplots
- Pie charts
- Bar charts

---

## Results & Insights
Some important insights discovered:
- Customers with month-to-month contracts are more likely to churn.
- Higher monthly charges can increase churn probability.
- Long-term customers show better retention.
- Certain payment methods are associated with higher churn rates.
